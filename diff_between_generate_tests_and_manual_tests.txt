Основные отличительные признаки кода, сгенерированного экспортом шагов из Selenium IDE и кода, написанного вручную, используя технологии PyTest+Selenium Webdriver

Код, сгенерированный экспортом шагов из Selenium IDE:
1. Содержит импорт неиспользуемых модулей.
2. Использует поиск элементов с помощью класса By.
3. Не содержит неявных/явных ожиданий.
4. Устанавливает заданное расширение окна в пикселях (.set_window_size(1865, 1037)).
5. Не содержит проверок с помощью assert.
6. Не содержит проверок визуального подтверждения правильности выполняемых действий, которые могут быть добавлены вручную. 
Например, что после нажатия на кнопку/ссылку загрузилась страница, содержащая такие-то элементы.
7. Содержит исключительно последовательность операций, выполняемых на сайте: клик по элементу, ввод текста в поле.
